# Change log: "All" inbox filter pill colour

Implements [plans/20260621_003157_all-pill-colour.md](../plans/20260621_003157_all-pill-colour.md).

## What changed

- `app/src/main/java/in/sreerajp/sms_sentry/ui/SmsOrganizerUi.kt`
  — in the inbox filter `pillColor` `when` (~line 1210), changed the **All**
  (`else`) branch from `MaterialTheme.colorScheme.primary` to
  `MaterialTheme.colorScheme.onSurfaceVariant`.

## Why

The **All** pill (`primary`, Lavender purple `#6F57AB`) was visually
indistinguishable from the **Personal** pill (`categoryColor("Personal")` violet
`#7A59C3`). The new neutral grey (`onSurfaceVariant`, per-scheme `TextMute`) is
distinct from all four category colours in both light and dark schemes.

## Scope

- One-line change; category pill branches untouched.
- No tests reference this colour; filtering behaviour unchanged.
