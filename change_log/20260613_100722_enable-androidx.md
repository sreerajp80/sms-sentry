# Enable `android.useAndroidX`

Implements [plans/20260613_100722_enable-androidx.md](../plans/20260613_100722_enable-androidx.md).

## What changed

- **gradle.properties** — added `android.useAndroidX=true` (with an explanatory comment) after
  the `kotlin.compiler.execution.strategy` line.

## Why

The AGP build was failing because AndroidX dependencies (`androidx.room:room-compiler:2.7.0`
and transitives) were on the classpath without `android.useAndroidX` enabled.

## Not done (intentionally)

- `android.enableJetifier` was not added — all dependencies are already AndroidX, so Jetifier
  rewriting is unnecessary.
