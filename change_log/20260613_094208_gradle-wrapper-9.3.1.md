# Change log: Update Gradle wrapper to 9.3.1

Implements plan: `plans/20260613_094208_gradle-wrapper-9.3.1.md`

## What changed

- `gradle/wrapper/gradle-wrapper.properties` — updated `distributionUrl`:
  - From: `https\://services.gradle.org/distributions/gradle-9.0-milestone-1-bin.zip`
  - To:   `https\://services.gradle.org/distributions/gradle-9.3.1-bin.zip`

No other properties were modified.

## Why

Android Studio Otter | 2025.2.1 with AGP 9.1.1 required a minimum stable Gradle
of 9.3.1; the wrapper previously pointed at the pre-release `9.0-milestone-1`,
blocking project sync/import.

## Verification

- Edit confirmed in the wrapper properties file.
- Functional verification is via Android Studio re-import/re-sync, which will
  download Gradle 9.3.1 on first run (no global Gradle install available for a
  CLI build check).
